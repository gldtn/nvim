# gldtn/nvim

My personal Neovim configuration.

## Features

- lazy.nvim package manager
- LSP config w/mason, mason-tool-installer, nvim-lspconfig
- conform formatting
- lualine statusline
- fzf-lua for fuzzy finding

## Themes

- Catppuccin
- Tokyonight
- Cyberdream

## Requirements

- Neovim v0.12
- Ghostty 1.2+ (Theme switching also changes Ghostty themes)

## About This Version

The goal of this setup is to stay lightweight, fast, and clean. Use only essentials plugins.
